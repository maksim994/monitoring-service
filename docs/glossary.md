# Glossary

| Термин | Значение |
| --- | --- |
| Organization | Tenant аккаунта, владеющий пользователями, сайтами, тарифом и настройками |
| Project | Группа сайтов внутри организации, обычно проект клиента интегратора |
| Client | Клиент интегратора, к которому могут относиться один или несколько проектов |
| Site | Мониторируемый сайт на 1С-Битрикс или коробочный портал Битрикс24 |
| Site key | Ключ подключения Bitrix-модуля к backend API |
| Bitrix module | Устанавливаемый модуль `vendor.monitoring`, собирающий внутренние метрики |
| Collector | Компонент модуля, собирающий конкретный набор технических данных |
| Heartbeat | Регулярное событие, подтверждающее, что модуль работает и доступен backend |
| Metric | Измеримое значение: свободный диск, версия PHP, число просроченных агентов |
| Event | Дискретное событие: найден последний бэкап, ошибка collector, изменение состояния |
| Check | Логическая проверка: uptime, SSL expiry, disk low, backup stale |
| Probe node | Stateless-нода, выполняющая внешние HTTP/SSL/DNS проверки |
| Probe result | Результат внешней проверки с evidence, временем ответа и ошибкой |
| Alert rule | Правило, которое превращает metric/event/probe result в инцидент |
| Fingerprint | Детерминированный ключ проблемы для дедупликации инцидентов |
| Incident | Зафиксированная проблема или риск, требующий внимания пользователя |
| Evidence | Данные, подтверждающие проблему: код ответа, ошибка SSL, значение метрики |
| Debounce | Задержка/подтверждение перед открытием или закрытием инцидента |
| Maintenance window | Окно обслуживания, в котором новые инциденты не создаются |
| Mute | Временное подавление уведомлений или конкретной проверки |
| Notification channel | Канал доставки уведомлений: Telegram, email, webhook |
| Delivery | Одна попытка отправки уведомления в канал |
| Module-only | Локальный режим модуля без SaaS/backend |
| Self-hosted | Развёртывание backend и probe-ноды в инфраструктуре клиента |
| Tenant scope | Обязательная фильтрация данных по organization/site во всех запросах и задачах |
