# Getting Started

## Первый Сайт

1. Откройте кабинет: http://localhost:13000
2. Войдите демо-аккаунтом:
   - Email: `demo@monitoring.local`
   - Password: `Demo123456`
3. Нажмите «Add site» или используйте уже созданный demo site.
4. Скопируйте `site_id` и `api_secret`.
5. Установите Bitrix-модуль.
6. Введите API URL `http://localhost:18080`, `site_id` и `api_secret`.
7. Нажмите «Проверить подключение».
8. Отправьте тестовый heartbeat.
9. Вернитесь в кабинет и убедитесь, что сайт подключён.

Подробнее о demo-аккаунте: `../development/demo-account.md`.

## Что Проверить После Подключения

- Последний heartbeat обновляется.
- Версия модуля отображается.
- Базовые проверки включены.
- Telegram или email канал настроен.
- Тестовое уведомление доставлено.

## Ожидаемое Время

Подключение первого сайта должно занимать не больше 15 минут.
